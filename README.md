# Mis 15 Esmeralda - Landing Page

Landing page para la celebración de quince años de Esmeralda.

## Características

- Contador regresivo hasta la fecha del evento
- Información de fecha y lugar
- Sección de Instagram para compartir fotos
- Formulario de confirmación de asistencia con envío a WhatsApp
- Música de fondo (I Gotta Feeling - Black Eyed Peas)
- Diseño mobile-first responsive

## Tecnologías

- HTML5
- CSS3
- JavaScript (Vanilla)
- Font Awesome (iconos)
